INSPECTION APP 

Here's brief description of the mobile app that helps inspectors conduct vessel inspections using their mobile devices

SIGN UP FEATURES

Conduct Inspections

Use the application to conduct vessel inspections on fisheries vessels in port.

PHOTOS & GPS
Photos and GPS locations can also be taken and the data is synced directly to the cloud when the mobile phone is in range of an internet connection. 

OFFLINE & CLOUD SYNC
This mobile application works offline and the data is then synced to the cloud when an internet connection is established.

Manage Data

The application can be used by multiple MCS Officers offline to manage information about vessel inspections.

As this data is synced to the cloud the information can then be synced back to the other MCS Officers. That way all officers have a holistic view of the inspections that are going on in near real-time.

Map Data

Due to the use of GPS location the mobile application will track where each inspection was carried out from so you know exactly who conducted the inspection, as well as when and where they were when it was being carried out.
